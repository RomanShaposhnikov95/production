теперь мы не можем больше делать общий тест в github actions так как там есть лимит и нужно оплатить.
Для этого мы будем использовать альтернативу https://typicode.github.io/husky/
устанавливае npx husky init
Теперь у нас установилась папочка в корне .husky
И в файле pre-commit мы устанавливаем все тесты которые должны запуститься
    npm run build:prod
    npm run lint:ts
    npm run lint:scss
    npm run test:unit

И тепер когда мы делаем commit у нас сбоку появляется tooltip c предупреждением об ошибках и мы можем клинуть по нему и увидеть в чем проблемма.
